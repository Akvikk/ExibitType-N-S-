# ExibitL 2.1

This contains the unified, offline-ready version of the application. 
Built as a Progressive Web Application (PWA), it can be installed locally and runs robustly without an active internet connection.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the development server (or serve via build):
   `npm run dev`
   *Or for offline-test building:*
   `npm run build && npx serve dist`
